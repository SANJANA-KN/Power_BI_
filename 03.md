# Data Cleaning using Power Query

## Introduction
Power Query is used to clean, transform, and prepare raw data before visualization.

---

# Data Cleaning Tasks
- Remove duplicates  
- Handle missing values  
- Change data types  
- Rename columns  
- Filter rows  

---

# Why Power Query
- Saves time  
- Automates transformations  
- Improves data quality  

---

# Common Transformations
- Split columns  
- Merge columns  
- Replace values  
- Trim spaces  

---

# Key Takeaway
Power Query ensures clean and structured data for accurate reporting in Power BI.
