# Data Sources and Import in Power BI

## Introduction
Power BI allows importing data from multiple sources for analysis and reporting.

---

# Data Sources
- Excel Files  
- CSV Files  
- SQL Databases  
- Web APIs  
- Cloud Services (Azure, Google Sheets)  

---

# Import Methods
- Get Data option  
- Direct Query  
- Live Connection  

---

# Steps to Import Data
1. Open Power BI Desktop  
2. Click Get Data  
3. Select source  
4. Load or transform data  

---

# Key Takeaway
Power BI supports multiple data sources, making it flexible for real-world business data integration.
