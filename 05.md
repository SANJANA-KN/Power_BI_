# DAX Basics in Power BI

## Introduction
DAX (Data Analysis Expressions) is a formula language used in Power BI for calculations.

---

# Basic DAX Functions
- SUM  
- AVERAGE  
- COUNT  
- MIN  
- MAX  

---

# Example
```DAX
Total Sales = SUM(Sales[Amount])
```

---

# Measures vs Columns
- Measures: Dynamic calculations  
- Columns: Static calculations  

---

# Key Takeaway
DAX is used to create powerful calculations in Power BI reports.
