# Data Modeling in Power BI

## Introduction
Data modeling is the process of creating relationships between tables to structure data efficiently.

---

# Types of Tables
- Fact Table (Transactions)  
- Dimension Table (Details)  

---

# Relationships
- One-to-Many  
- Many-to-One  
- One-to-One  

---

# Star Schema
Best practice model:
- Central fact table  
- Surrounding dimension tables  

---

# Key Takeaway
Good data modeling improves performance and makes reporting accurate and efficient.
