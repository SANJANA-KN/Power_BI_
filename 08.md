# Charts Types in Power BI

## Introduction
Power BI provides multiple chart types for different data analysis needs.

---

# Chart Types
- Column Chart  
- Bar Chart  
- Line Chart  
- Area Chart  
- Pie Chart  
- Donut Chart  

---

# Usage
- Comparison → Bar/Column  
- Trends → Line Chart  
- Proportion → Pie/Donut  

---

# Key Takeaway
Choosing the right chart improves clarity and insight delivery.
